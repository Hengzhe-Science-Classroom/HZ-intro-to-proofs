# Introduction to Proofs · Interactive

**Learning to write rigorous mathematical arguments**

**Status:** Planned

Part of [Hengzhe's Science Classroom](https://hengzhe-science-classroom.github.io/hub/) · Tier 3 · Undergraduate

## What You'll Learn

- Logic and propositions
- Direct proof and contrapositive
- Proof by contradiction
- Mathematical induction
- Set theory proofs
- Relations, functions, and cardinality

## References

- Richard Hammack, *Book of Proof* (3rd ed., Virginia Commonwealth University)
- Daniel J. Velleman, *How to Prove It* (3rd ed., Cambridge)
- Kevin Houston, *How to Think Like a Mathematician* (Cambridge)

## Features (when built)

- **Interactive visualizations** — every concept comes alive with draggable, animated demos
- **KaTeX math rendering** — rigorous notation throughout
- **Exercises with hints & solutions** — from computational to proof-based
- **Progress tracking** — saved locally in your browser
- **Zero dependencies** — just open the link and learn
